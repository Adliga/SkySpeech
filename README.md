# SkySpeech

Интерфейс для синтеза и клонирования голоса. Фронтенд на Vue 3 + Pinia + Axios.

## Запуск

```sh
npm install
npm run dev
```

Сборка:

```sh
npm run build
```

## Стек

- Vue 3 (Composition API, `<script setup>`)
- Pinia — сторы
- Axios — запросы к бэкенду (`localhost:8000`)
- Vite — сборка
